Projet de jeu en NSI

# Pendu #

Mode joueur solo et multijoueur                     
  --> Trouver un dictionnaire sur Google

Fonctions à coder : 
1. Choix du mot
2. Dessin progressif
3. Si la lettre est correcte
4. Affichage du mot avec les lettres découvertes et pas bonnes
5. Classement des joueurs
